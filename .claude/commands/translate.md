Translate GrowZen content from Chinese (zh) to English (en) and Japanese (ja).

The user will provide either:
- A specific file path (e.g. content/zh/principles/virtue-before-power.md)
- A section name to translate all missing files
- "all" to find and translate all zh files that don't have en/ja counterparts

Steps:
1. Read the source zh file carefully
2. Identify the Buddhist teaching being expressed
3. For English translation:
   - Translate meaning, not words — adapt idioms so they work in English
   - Use universal examples (not China-specific cultural references for en)
   - Keep the three-part structure (Observation → Principle → Application)
   - Short sentences. No jargon.
   - Save to content/en/<section>/<same-slug>.md

4. For Japanese translation:
   - Preserve Buddhist terminology in Japanese (慈悲, 無常, 正念, etc.)
   - Keep the haiku-like precision of the Chinese original where possible
   - Save to content/ja/<section>/<same-slug>.md

5. After translating, run writing-check on the en version to ensure universal accessibility

Do not translate literally — translate in spirit.
